# External Expert Skills Notice

This company plugin bundles selected expert skills from `sickn33/antigravity-awesome-skills` so company users can use the workflow out of the box.

- Upstream: https://github.com/sickn33/antigravity-awesome-skills
- Observed upstream release label: `V12.3.0`
- Repository code license: MIT
- Original non-code content license: CC BY 4.0 unless upstream states otherwise

Bundled expert skills are treated as managed dependencies. They are checked by `scripts/expert_dependencies.py` during plugin install and project bootstrap. Future updates must go through company skill maintenance and security review rather than silent hub replacement.

# 将模板安装到公司项目

1. 将 `AGENTS.md` 复制到项目根目录，或把其中章节合并到已有项目说明中。
2. 将 `specs/` 复制到项目根目录。
3. 填写 `specs/global/INDEX.md`，补充项目概览和常用命令。
4. 每个功能创建 `specs/features/<feature>/`，只复制需要的模板。
5. 先用一个小功能跑通流程，再决定是否作为团队强制流程。

建议试点成功标准：

- 一个功能可以从需求追溯到验证证据。
- 一个 bugfix 记录了根因和回归验证。
- Codex 从索引开始读取，不加载无关 spec。
- 团队会删除长期为空的模板字段。

# Install This Template Into a Company Project

1. Copy `AGENTS.md` into the project root, or merge its sections into the existing project instructions.
2. Copy `specs/` into the project root.
3. Fill `specs/global/INDEX.md` with the project snapshot and commands.
4. For each feature, create `specs/features/<feature>/` and copy only the templates needed.
5. Run one small feature through the workflow before making it mandatory for the team.

Recommended pilot success criteria:

- One feature can be traced from requirement to verification.
- One bugfix records root cause and regression evidence.
- Codex starts by reading the index and does not load unrelated specs.
- The team removes any template section that repeatedly stays empty.
